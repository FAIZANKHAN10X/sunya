# Roadmap — Sunya

## Phase 1 — MVP Foundation

**Status: Complete**

- Next.js App Router project structure under `src/`
- Loading screen, sticky header, placeholder menu
- Seven data-driven full-viewport sections
- Minimal footer
- TypeScript strict mode and Tailwind CSS v4
- Theme support (light / dark / system)
- Root documentation (overview, structure, guidelines, roadmap)

## Phase 2 — Content Development

**Status: Complete (placeholder content)**

- Believable placeholder copy across all seven sections
- Clear information architecture: Hero, Philosophy, Practice, Benefits, Journey, Community, Contact
- Accessible heading hierarchy and semantic section structure
- Typed content model ready for future real copy or CMS migration

## Phase 3 — Visual Design

**Status: In progress (structure & system complete; imagery later)**

### Done

- Restrained visual system (tokens, spacing, typography, radius, containers)
- Refined header, menu, section layouts, and footer
- Soft surfaces, subtle borders, and calm premium layout patterns
- Responsive polish for stacked section compositions

### Remaining

- Brand imagery and photography direction
- Optional deeper layout variants without breaking the stacked model
- Final visual QA against real brand assets when available

## Phase 4 — Animation & Interaction

**Status: Not started**

- Introduce purposeful motion only where it supports clarity and calm
- Evaluate animation approach (CSS-first preferred; libraries only if justified)
- Implement real navigation behavior for menu items
- Wire CTAs and contact actions to real destinations or forms
- Consider scroll-linked or entrance effects carefully and sparingly

## Phase 5 — Production Launch

**Status: Not started**

- SEO metadata, Open Graph, and sitemap
- Performance pass (fonts, images, Core Web Vitals)
- Accessibility audit
- Analytics and privacy considerations
- Production hosting, domain, and monitoring
- Final content freeze and launch checklist
